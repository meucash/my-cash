# MeuCash - Controle Financeiro

Sistema simples de controle financeiro feito em Python.

## Como usar

1. Certifique-se de ter o Python 3 instalado
2. Execute o programa:

```bash
python main.py
```

3. Digite seu nome e veja seu saldo

## Funcionalidades

- Saudacao personalizada
- Calculo de saldo (entradas - saidas)
- Alerta de saldo positivo, zerado ou negativo
